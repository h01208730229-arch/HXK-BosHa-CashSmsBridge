# Cash SMS Bridge — Vodafone Cash

التطبيق يقرأ إشعارات SMS الواردة ويرسلها إلى بوابة Laravel.

## البناء عبر GitHub Actions
ارفع **محتويات هذا المجلد مباشرة إلى جذر المستودع**. ملف البناء موجود في:

`.github/workflows/android.yml`

ثم Actions > Build Android APK > Run workflow.

## إعداد التطبيق
- رابط السيرفر: `https://ddd.omnione.ink/api/cash-sms/receive`
- مفتاح الحماية: `bb123bb`

زر الاختبار يعرض كود HTTP ورد السيرفر لتسهيل تشخيص أي مشكلة.
